## Installation

FFText is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'FFText'
```

## Feature

- [x] 富文本基础使用：文字色、背景色、字体、下划线...
- [x] 图文、插入自定义控件 
- [x] 点击高亮 
- [x] 自动识别链接、邮箱地址、手机号、自定义识别规则
- [x] 排空区域
- [x] 约束布局
- [x] 自定义截断符
- [x] 文本垂直布局      
- [ ] 模糊约束支持
- [ ] 长按选则文本内容
- [ ] 异步渲染
